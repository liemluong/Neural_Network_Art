Background

     

AI Fashion is a capstone project for the University of Washington Master of Science in Data Science program. The capstone project gave our team of students the opportunity to create a data science project that has never been before approached in this way. This capstone project was presented at the Capstone Event hosted by the University. To learn more about the University of Washington Master of Science in Data Science capstone project, visit this link.

     

 

Project Deliverables

 

Interactive Website

Project Repository

     

The project repository is hosted on GitHub, and includes the source code for both the component 1 model and the component 2 model. We also provide access to our entire training dataset, and final pattern results. The source code for this website and how to run it locally is also available.

https://github.com/liemluong/Neural_Network_Art

 

 

Future Works
